AutoYaST integration tests

Just some notes about integration tests

November, 2015

How it works

  • Grabs a ISO image (for example SLE 12 SP1 RC3)
  • Adds latest YaST packages from OBS (also custom packages can be added)
  • Runs the installation using a given profile
  • Runs a set of tests on the installed system

Some features

  • Include latest YaST packages (or anything you want)
  • Support for two virtualization technologies
  • Writing tests is pretty straighforward
  • Upgrade scenario is supported
